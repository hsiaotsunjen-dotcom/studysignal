# StudySignal homepage

Mobile-first dark UI for StudySignal — Next.js (App Router), React, and Tailwind CSS. Frontend only; no API.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 15
- React 19
- Tailwind CSS 3
- `lucide-react` icons
